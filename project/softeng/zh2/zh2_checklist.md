## Checklist a második ZH-hoz



❶ API Controllerek létrehozása: létre kell tudni hozni egy alkalmazást az ASP. NET sablon alapján, és ebben:

- Beállítani a statikus tartalom kiszolglását a `wwwroot` mappából
- Elkészíteni az ORM-et egy terszőleges adatbázihoz
- API controller-t kell tudni létrehozni, mely a `GET`, `POST` ill. `DELETE` metódusokon keresztül megvalósítja a CRUDE műveleteket

❷ HTML / JS techonlógiával weblapot kell tudni létrehozni, amely a előző pontban létrehozott API végpontok hívásával 

- megjeleníti egy adattábla tartalmát
- Lehetőséget biztosít új rekord felvételére. 

Idegen kulcsokat és legördülő dobozokat nem kell tudni kezelni. 

❸`ClosedXML` vagy más, szabadon választott Nu-Get csomag segítségével adatbázis tábla tartalmát Excel állaományba kell tudni írni WinForms alkalmazásból. 

❹ ExcelDNA technológiával beépülőt kell tudni létrehozni, amely pl. képes betölteni egy adatbázistáblát spill array-ként. 

❺ Minden, ami az előző ZH-ban volt, de nem ezeken lesz a hangsúly.
